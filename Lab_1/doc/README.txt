TASK: Лабораторная работа №1 — Цветовые модели

AUTHOR: Шушкевич Дмитрий, 2 курс 12 группа ФПМИ

ROAD MAP:
	Написать приложение/веб-приложение, позволяющее взаимодействовать с существующими цветовыми моделями.

	Градиентный квадрат по нажатию клавиши позволяет выбрать цвет из градиентной палитры и задать текущим. 
		Ползунки, значения в полях ввода автоматически изменяются;
	Одноцветный прямоугольник отображает текущий цвет;
	Ползунки позволяют изменять текущий цвет более плавно;
	Dropdown меню необходимое для переключения между цветовыми моделями.

USED TEHCNOLOGIES:
	Windows Forms с использованим C#.

P.S. 
	Приложение можно заупсить из папки code\bin\Release\netcoreapp3.1, двойным кликом по файлу "ColorConverter.exe".
